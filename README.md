# Python Exercises

Practice exercises covering Python fundamentals, file handling, and Excel automation with openpyxl.

## Files

| File | Description |
|------|-------------|
| `calculate-lcm.py` | Calculates the Least Common Multiple (LCM) of three integers |
| `multiples-list.py` | Reads 10 numbers and counts multiples of a given divisor |
| `string-pyramid.py` | Prints a word as a growing pyramid, one letter at a time |
| `vector-sum.py` | Reads two lists of 15 numbers and creates a third with their element-wise sum |
| `hangman.py` | Interactive hangman game with random word selection |
| `file-operations.py` | Writing, appending, and reading text files in different modes |
| `file-search.py` | Searches for a word in a text file |
| `file-append.py` | Demonstrates appending and overwriting file content across multiple steps |
| `excel-read.py` | Reads all sheets from an Excel file and duplicates them |
| `excel-create.py` | Creates a new Excel workbook with multiple sheets, data, and a formula |
| `excel-population-data.py` | Extracts population data from Excel to calculate totals and averages |

## Requirements

```bash
pip install openpyxl
```

## How to run

```bash
python <filename>.py
```
